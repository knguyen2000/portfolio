# Project: Zero-Cost Verifiable Portfolio

Description:
This project demonstrates how to build a hallucination-resistant chatbot using purely local python algorithms.

Key Features:

1. **Verbatim Provenance**: Every claim is linked to a source document.
2. **Zero Cost**: Runs on Streamlit Community Cloud and Gemini 1.5 Flash Free Tier.
3. **Trace Engine**: Custom greedy matching algorithm to find maximal exact matches.

Technologies:

- Streamlit
- Google Gemini
- Python Suffix Arrays (simulated)
